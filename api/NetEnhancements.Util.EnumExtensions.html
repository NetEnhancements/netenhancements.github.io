GetDisplayName(Enum)

Read the Name property from an enum member.

When the member can't be found by name, or when none of the attributes is found, the passed enum member's ToString() will be returned.

public static string GetDisplayName(this Enum enumMember)

enumMember Enum

Examples

enum Foo
{
    [Display(Name = "Foo-1")]
    Foo1
}

// will contain "Foo-1".
var foo1Name = Foo.Foo1.GetDisplayName();
